# Bus Price Tracker

A mobile and web application that tracks real-time bus price fluctuations, helping travelers find the best last-minute deals for their journeys.

## Overview

Bus prices fluctuate significantly depending on demand, time until departure, and other factors. Last-minute bookings often yield unexpectedly low prices. This application helps users track bus prices from their preferred routes and sends alerts when prices drop below their target threshold.

## Features

- **Real-time Price Tracking**: Monitor bus prices across multiple routes
- **Price Alerts**: Get notified when prices drop below your target
- **Price History**: View historical price trends for routes
- **Multi-platform**: Works on iOS, Android, and Web
- **User Authentication**: Secure login and profile management
- **Settings Management**: Customize alert thresholds and preferences

## Tech Stack

### Backend
- Node.js with Express.js
- TypeScript for type safety
- PostgreSQL for data persistence
- JWT for authentication

### Frontend
- React Native for mobile apps (iOS/Android)
- React for web application
- Redux for state management
